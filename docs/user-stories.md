1. As a user, I can sign up for an account so I can save my plans.
2. As a user, I can log in so I can access my saved plans.
3. As a user, I can log out so others can’t access my account.
4. As a user, I can create a new study plan with a title and dates.
5. As a user, I can edit a study plan if my schedule changes.
6. As a user, I can delete a study plan I no longer need.
7. As a user, I can view a list of my plans so I can choose one quickly.
8. As a user, I can open a plan to see full details (sessions/tasks).
9. As a user, I can mark a plan as a public template to share with others.
10. As a user, I can browse a gallery of public templates created by other users.
11. As a user, I can view a template’s details before using it.
12. As a user, I can copy a template to create my own editable plan.
